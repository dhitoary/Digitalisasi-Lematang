# Folder Gambar UMKM

Folder ini berisi gambar-gambar produk dan usaha UMKM di Desa Lematang.

## Format Penamaan File

Gunakan format penamaan file yang konsisten:
- `nama-umkm.jpg` (huruf kecil, pisahkan dengan tanda hubung)
- Contoh: `warung-kita.jpg`, `baso-azam.jpg`

## Spesifikasi Gambar

- **Format**: JPG atau PNG
- **Ukuran**: Minimal 800x600 pixels
- **Rasio**: 4:3 atau 16:9 (landscape)
- **Ukuran File**: Maksimal 500KB (untuk optimasi loading)
