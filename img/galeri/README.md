# Folder Galeri Desa

Folder ini berisi gambar-gambar untuk galeri kegiatan desa.

## Format Penamaan:
- Gunakan nama deskriptif: `kegiatan-posyandu.jpg`, `gotong-royong.jpg`, dll.
- Atau gunakan nomor urut: `1.jpg`, `2.jpg`, `3.jpg`, dst.

## Contoh:
```
img/galeri/
├── posyandu-lansia.jpg
├── gotong-royong.jpg
├── musyawarah-desa.jpg
├── perayaan-17-agustus.jpg
└── pembangunan-jalan.jpg
```

Gambar-gambar ini akan ditampilkan di halaman Galeri Desa (`galeri.html`).
