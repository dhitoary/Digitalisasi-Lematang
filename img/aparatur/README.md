# Folder Aparatur Desa

Folder ini berisi foto-foto aparatur/pejabat desa untuk ditampilkan di halaman Pemerintahan.

## Format Penamaan:
Gunakan nama jabatan atau nama lengkap (huruf kecil, pisahkan dengan dash):

**Contoh:**
```
img/aparatur/
├── kepala-desa.jpg
├── sekretaris-desa.jpg
├── kaur-keuangan.jpg
├── kaur-umum.jpg
├── kaur-perencanaan.jpg
├── kasi-pemerintahan.jpg
├── kasi-kesejahteraan.jpg
├── kasi-pelayanan.jpg
├── kepala-dusun-1.jpg
├── kepala-dusun-2.jpg
└── kepala-dusun-3.jpg
```

## Spesifikasi Gambar:
- Format: JPG/PNG
- Ukuran yang disarankan: 400x400px (rasio 1:1)
- Background: Formal/seragam

Foto-foto ini akan ditampilkan di halaman struktur organisasi pemerintahan desa.
