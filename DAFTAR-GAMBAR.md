# Daftar Gambar yang Dibutuhkan

Untuk menampilkan modal dengan sempurna, siapkan gambar-gambar berikut di folder `img/`:

## Gambar UMKM

1. **keripik-pisang.jpg**
   - Ukuran: 800x600px (rasio 4:3)
   - Foto produk keripik pisang dalam kemasan
   - Format: JPG atau PNG

2. **beras-lokal.jpg**
   - Ukuran: 800x600px
   - Foto beras atau karung beras
   - Format: JPG atau PNG

3. **anyaman-bambu.jpg**
   - Ukuran: 800x600px
   - Foto produk kerajinan anyaman bambu
   - Format: JPG atau PNG

## Gambar Wisata

1. **wisata-pertanian.jpg**
   - Ukuran: 800x600px
   - Foto area persawahan atau pertanian
   - Format: JPG atau PNG

2. **wisata-alam.jpg**
   - Ukuran: 800x600px
   - Foto pemandangan alam atau sungai
   - Format: JPG atau PNG

3. **wisata-budaya.jpg**
   - Ukuran: 800x600px
   - Foto bangunan bersejarah atau aktivitas budaya
   - Format: JPG atau PNG

4. **wisata-industri.jpg**
   - Ukuran: 800x600px
   - Foto kawasan industri atau pabrik
   - Format: JPG atau PNG

## Tips Mengambil Foto

1. **Pencahayaan**: Ambil foto di siang hari dengan cahaya yang cukup
2. **Sudut**: Ambil dari sudut yang menarik dan menunjukkan detail produk/tempat
3. **Background**: Gunakan background yang bersih dan tidak ramai
4. **Resolusi**: Minimal 800x600px untuk kualitas yang baik
5. **Format**: JPG untuk file size lebih kecil, PNG untuk kualitas lebih baik

## Alternatif Jika Belum Ada Gambar

Jika belum memiliki foto, Anda bisa:

1. **Gunakan placeholder**: File akan menampilkan broken image hingga foto ditambahkan
2. **Download dari sumber free**: 
   - Unsplash.com
   - Pexels.com
   - Pixabay.com
3. **Buat gambar sederhana**: Gunakan Canva atau tool design lainnya

## Cara Upload Gambar

1. Siapkan file gambar dengan nama yang sesuai
2. Copy file ke folder `img/` di root project
3. Pastikan nama file sama persis dengan yang ada di `js/script.js`
4. Test dengan membuka `test-modal.html` atau `pariwisata.html`

## Optimasi Gambar (Opsional)

Untuk performa website yang lebih baik:

1. Compress gambar menggunakan:
   - TinyPNG.com
   - Squoosh.app
   - ImageOptim (Mac)
   
2. Target file size: < 200KB per gambar

3. Pertahankan kualitas visual yang baik
